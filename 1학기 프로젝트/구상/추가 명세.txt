지도 API 활용

주변 운동 시설 조회
: 주소를 입력하면, 해당 주소 기준 3km 반경 이내의 운동 시설을 보여준다.
/POST

운동 시설 평점순/거리순/조회순 정렬
: 정렬 버튼의 조건을 선택하면 해당 조건을 기준으로 정렬해준다.
/GET

운동 시설 상세 조회
: 보여준 운동 시설 중 하나를 선택하면 해당 운동 시설의 상세 정보를 보여준다.
/GET


쇼핑몰 기능

운동 용품 전체 조회
: 운동 용품 목록을 전체 조회
/GET

운동 용품 상세 조회
: 운동 용품의 상세 정보를 조회
/GET


나의 운동 일기장 기능

나의 운동 일기장 작성
: 오늘 나의 운동 일지를 작성할 수 있는 기능
/POST

운동 일기장 전체/상세 조회
: 전체/상세 운동 일지를 조회
/GET
